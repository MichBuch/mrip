How to test:
1. Scroll around this page (some images are "lazy" style).
2. Click the big green Run mrip (dev) button below — it runs the readable development version.
3. Or use the bookmarklet string from README.md.
4. To install the real bookmarklet, drag the green link in the prominent banner or the Install box below to your bookmarks bar. Re-drag after updates to refresh.

New in this version: Live search in the overlay, better scaled tiles, default unchecked, FS/DL badge, improved download fallback.
Browser support: Edge/Chrome/Brave/Opera = full (incl. folder picker). Firefox = excellent overlay + standard downloads (no picker).
